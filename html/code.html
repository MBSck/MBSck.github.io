Work projects (My Contribution)

• oimodeler (Contributor):

A modular modelling tool for optical interferometry.

• ppdmod (Owner):

An interferometric modelling tool, which I am step by step integrating into oimodeler (it will then be archived).

• matadrs (Owner):

The (modular) MATISSE Automated Data Reduction Software (Matadrs) is a tool written (with some code adapted from colleagues) to encompass all the individual steps and files needed to procure and reduce data generally and easily in as few steps as possible with the least knowledge required. To make it possible for seasoned MATISSE users to quickly and easily reduce data and giving others who are interested in MATISSE data the opportunity to do so as well.

• p2obp (Owner):

The Phase 2 OB Pipeline (p2obp) is made to streamline/automate the process of MATISSE observation preparation on p2 (can theoretically also be used for all other VLTI instruments, if templates are adapted).
