Code

• matadrs:

The (modular) MATISSE Automated Data Reduction Software (Matadrs) is a tool written to encompass all the individual steps and files needed to procure and reduce data generally and easily in as few steps as possible with the least knowledge required. To make it possible for seasoned MATISSE users to quickly and easily reduce data and giving others who are interested in MATISSE data the opportunity to do so as well.

• p2obp:

The Phase 2 OB Pipeline (p2obp) has been made to streamline/automate the process of MATISSE observation preparation on p2.
